# PolishForge

PolishForge is a Unity 6 game-feel framework for prototypes and game jam projects. It adds reusable feedback presets for camera shake, screen flash, hit stop, audio variation, VFX spawning, floating text, UI punch, and other presentation polish.

## Install

Keep the `Assets/_PolishForge` folder in the project. The runtime assembly is independent from JamForge.

## Core Usage

Add these services to a scene object, or let effects auto-create their service when first used:

- `FeedbackPlayer`
- `PolishSettings`
- `PolishCameraShake`
- `PolishAudioPlayer`
- `ScreenFlashOverlay`
- `ScreenFadeOverlay`
- `HitStopController`
- `VfxSpawner`
- `FloatingTextSpawner`
- `RumbleController`

## Create A FeedbackPreset

Use `Create > PolishForge > Feedback Preset`, then add effect assets such as `CameraShakeEffect`, `ScreenFlashEffect`, `HitStopEffect`, `AudioCueEffect`, `VfxSpawnEffect`, `ScalePunchEffect`, `ColorFlashEffect`, or `FloatingTextEffect`.

## Trigger From Code

```csharp
FeedbackPlayer.PlayGlobal(enemyHitPreset, new FeedbackContext
{
    Source = player,
    Target = enemy,
    Position = enemy.transform.position,
    Intensity = 1f,
    Amount = damage
});
```

## Trigger From UnityEvent

Add `FeedbackTrigger` to any GameObject, assign a preset, then call `Play()` from a button or animation event.

## Audio And VFX

Create audio cues with `Create > PolishForge > Audio Cue`. Create VFX cues with `Create > PolishForge > VFX Cue`.

## UI Juice

Add `JuicyButton` to a Unity UI `Button` for hover, press, sound, and optional feedback preset support. Use `NumberCounter`, `UIPunch`, `UIShake`, `UIFadeSlide`, and `ToastManager` for HUD polish.

## Accessibility

`PolishSettings` stores intensity and enable/disable preferences in PlayerPrefs. Camera shake, screen flash, hit stop, rumble, slow motion, and floating text respect these settings.

## Debug Panel

Add `PolishDebugPanel` to a scene and press `F2` in the editor or a development build.

## Checklist

Open `Tools > PolishForge > Polish Checklist` before submitting a jam build.

## JamForge Integration

Core PolishForge does not depend on JamForge. A bridge can be added under `Scripts/Integrations/JamForge` when JamForge is present.

## Known Limitations

Default preset assets and demo prefabs are not generated by this first code pass. Create presets from the Unity Create menu and assign effects manually.
